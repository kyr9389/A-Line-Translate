# A-Line Translate 

한 줄 번역기는 구글 크롬 확장 프로그램으로 지원되는 간편한 번역 프로그램입니다.  
복사 붙여넣기 과정 없이 드래그만으로 손쉽게 영단어 또는 문장을 번역할 수 있습니다.

API 통신을 위해 [한 줄 번역기 API](https://github.com/kyr9389/A-Line-Translate-api) 프로젝트를 이용합니다.

## 기술스택

- node.js
- react.js
